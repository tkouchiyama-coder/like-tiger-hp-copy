 写真の差し替え方法（とてもかんたんです）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

この「images」フォルダに、下の名前で写真ファイルを入れるだけで、
ホームページの「写真が入ります」のグレー部分に自動で表示されます。
（プログラムの修正は一切不要です）

■ 入れるファイル名と場所

【セクション1：トップ】
  hero-main.jpg    … メインの大きな写真（営業中の姿・旅の写真など）
  hero-sub1.jpg    … サブ写真1（小さめ・ポラロイド風に表示）
  hero-sub2.jpg    … サブ写真2

【理念・ビジョンページ（philosophy.html）】
  vision-team.jpg  … チーム・仲間の写真

【セクション4：自己紹介】
  profile-main.jpg … 代表 河内山さんのメイン写真
  profile-sub1.jpg … 旅の写真1（世界一周・ウユニ塩湖など）
  profile-sub2.jpg … 旅の写真2

■ 注意点
  ・ファイル名は半角英数字で、上と完全に同じ名前にしてください
  ・形式は .jpg（.png でも可。その場合は index.html 内の
    「data-photo」のファイル名の末尾を .png に変えてください）
  ・写真を入れたあと、ブラウザで再読み込み（⌘+R）すると反映されます
